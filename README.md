### About

Embedded Systems Intern and robotics engineer in training, focused on the intersection of **low-level hardware** and **autonomous software stacks**. I build things end-to-end: schematic &rarr; PCB &rarr; firmware &rarr; ROS 2 node &rarr; simulation &rarr; field deployment. Interested in roles in **autonomous mobile robotics**, **perception**, and **sim-to-real** pipelines.

### Focus Areas

- **Robotics Middleware** &mdash; ROS 2 (Humble), DDS, `tf2`, lifecycle nodes, custom message/action design
- **Autonomy** &mdash; Nav2 stack, SLAM (slam_toolbox, Cartographer), sensor fusion, path planning
- **Simulation** &mdash; NVIDIA Isaac Sim, Isaac Lab, Gazebo, URDF/SDF modeling
- **Embedded** &mdash; ESP32, STM32, ARM Cortex-M, real-time firmware, sensor interfacing (I2C/SPI/UART/CAN)
- **Hardware** &mdash; Multi-layer PCB design, power electronics basics, motor control, schematic review
- **Perception &amp; Embedded AI** &mdash; OpenCV, PyTorch, model deployment on edge devices

### Currently

- Working as an **Embedded Systems Intern** &mdash; firmware, hardware bring-up, and product integration
- Deepening expertise in **ROS 2 Nav2**, **SLAM**, and **Isaac Sim / Isaac Lab** for robot learning workflows
- Building toward a portfolio of **reproducible, documented robotics projects** (sim + real hardware)
