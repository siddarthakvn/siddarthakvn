## &nbsp;&#9679;&nbsp; About

I build robots end-to-end &mdash; **schematic &rarr; PCB &rarr; firmware &rarr; ROS 2 node &rarr; simulation &rarr; field**. Currently an **Embedded Systems Intern** working on microcontroller firmware and hardware bring-up, while developing a robotics portfolio around autonomous navigation, SLAM, and sim-to-real pipelines in NVIDIA Isaac Sim. Open to roles in **autonomous mobile robotics**, **robot perception**, and **embedded AI**.
